# **App Name**: NutriSnap

## Core Features:

- Food Recognition: Utilize the Gemini API to analyze uploaded food images or descriptions and identify food items.
- Nutritional Analysis: Provide detailed caloric and macronutrient breakdowns (calories, protein, carbs, fats) for each identified food item.
- Food Log: Maintain a scrollable, chronological log of food entries, displaying up to the past 10 days.
- Daily Summary: Present a daily summary of total caloric and macronutrient intake, updating in real time as new entries are added.

## Style Guidelines:

- Primary color: Fresh green (#64B87A), for a natural and healthy feel.
- Background color: Light greyish-green (#F0FAF2).
- Accent color: Yellow-green (#B2DFA3) for subtle highlights and CTAs.
- Body and headline font: 'PT Sans', a modern sans-serif with warmth suitable for all text.
- Use clean, minimalist icons to represent different food groups and nutrients.
- Maintain a clean and intuitive layout with clear section headings and a focus on readability.
- Use subtle animations (e.g., smooth transitions, loading indicators) to enhance the user experience.
